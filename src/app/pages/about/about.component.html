Construção e Descrição do Projeto:

Projeto realizado com o intuito de realizar um CRUD básico, para testar os conhecimentos do framework de Front End Angular, colocando os principais conceitos e funcionalidades aprendidas até aqui. Para os dados fornecidos, vieram provenientes da nossa API local, criada com a lib do json-server.
O sistema de cadastro possui os seguintes campos:

  • Id
  • Nome do Funcionário
  • Cargo
  • Idade
